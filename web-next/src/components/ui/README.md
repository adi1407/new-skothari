# UI Primitives

Use this folder for design-system level reusable building blocks.

Examples:

- `Button`
- `Card`
- `Badge`
- `Chip`
- `Input`
- `Tabs`
- `Modal`
- `Skeleton`

Rules:

- Keep UI components domain-agnostic.
- No API calls inside primitives.
- No route-specific SEO logic.
- Prefer props-driven styling with shared tokens from global CSS.
