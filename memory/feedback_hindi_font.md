Always use **Noto Sans Devanagari** for Hindi/Devanagari text in the news-kothari project.

**Why:** User confirmed this is the correct font for Hindi readability.

**How to apply:** CSS variable `--font-hi: "Noto Sans Devanagari", sans-serif;` — use for all body text, headlines, and UI labels that display Hindi content. Pair with "Inter" for English/numeric elements.
